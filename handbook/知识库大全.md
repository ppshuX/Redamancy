## 公共基础知识

### 一、数据结构与算法

#### 1. 算法的基本概念
**Q: 什么是算法？**
A: 算法是指解题方案的准确而完整的描述，是一系列解决问题的清晰指令。

**Q: 算法的四个基本特征？**
A: 
- 可行性
- 确定性
- 有穷性
- 拥有足够的情报（输入/输出）

**Q: 算法的时间复杂度和空间复杂度的定义？**
A: 
- 时间复杂度：执行算法所需要的计算工作量
- 空间复杂度：执行算法所需要的内存空间

#### 2. 数据结构的基本概念
**Q: 什么是数据结构？**
A: 数据结构是指相互有关联的数据元素的集合。

**Q: 数据的逻辑结构和存储结构的区别？**
A: 
- 逻辑结构：数据元素之间的逻辑关系（集合、线性、树形、图形）
- 存储结构：数据在计算机中的存储方式（顺序、链式、索引、散列）

#### 3. 线性表
**Q: 线性表的特点？**
A: 
- 只有一个根结点
- 只有一个终端结点
- 除根结点和终端结点外，其他结点只有一个前件和一个后件

**Q: 线性表的顺序存储和链式存储的优缺点？**
A: 
- 顺序存储：查找方便，插入删除效率低
- 链式存储：插入删除方便，查找效率低

#### 4. 栈和队列
**Q: 栈的特点？**
A: 先进后出（FILO）或后进先出（LIFO）

**Q: 队列的特点？**
A: 先进先出（FIFO）

**Q: 栈和队列的应用场景？**
A: 
- 栈：函数调用、递归、括号匹配
- 队列：排队系统、广度优先搜索

#### 5. 树和二叉树
**Q: 二叉树的性质？**
A: 
- 性质1：在二叉树的第k层上，最多有2^(k-1)个结点
- 性质2：深度为m的二叉树最多有2^m - 1个结点
- 性质3：度为0的结点（叶子结点）比度为2的结点多1个

**Q: 二叉树的遍历方式？**
A: 
- 前序遍历：根→左→右
- 中序遍历：左→根→右
- 后序遍历：左→右→根
- 层序遍历：从上到下，从左到右

#### 6. 查找技术
**Q: 顺序查找和二分查找的时间复杂度？**
A: 
- 顺序查找：O(n)
- 二分查找：O(log₂n)

**Q: 二分查找的条件？**
A: 线性表必须是有序表

#### 7. 排序技术
**Q: 常见排序算法的时间复杂度？**

| 排序方法 | 平均时间复杂度 | 最坏情况 |
|---------|--------------|---------|
| 冒泡排序 | O(n²) | O(n²) |
| 快速排序 | O(nlog₂n) | O(n²) |
| 简单插入排序 | O(n²) | O(n²) |
| 简单选择排序 | O(n²) | O(n²) |
| 堆排序 | O(nlog₂n) | O(nlog₂n) |
| 希尔排序 | O(n^1.5) | - |

### 三、软件工程基础

#### 1. 软件定义与软件特点
**Q: 软件的定义？**
A: 软件是与计算机系统的操作有关的程序、数据和相关文档的集合。

**Q: 软件的特点？**
A: 
- 是逻辑实体，不是物理实体
- 是脑力劳动的产物
- 不会磨损，但可以退化
- 开发成本高，维护成本更高

#### 2. 软件生命周期
**Q: 软件生命周期的三个时期？**
A: 
- 软件定义期：问题定义、可行性研究、需求分析
- 软件开发期：概要设计、详细设计、实现、测试
- 软件维护期：使用、维护

**Q: 软件维护的分类？**
A: 
- 改正性维护：修正错误
- 适应性维护：适应环境变化
- 完善性维护：增强功能
- 预防性维护：为将来维护做准备

#### 3. 软件设计
**Q: 模块独立性的两个定性指标？**
A: 
- 耦合性：模块间相互依赖的程度（低耦合好）
- 内聚性：模块内部各元素彼此结合的紧密程度（高内聚好）

**Q: 常见的耦合类型（从低到高）？**
A: 
- 无直接耦合
- 数据耦合
- 标记耦合
- 控制耦合
- 外部耦合
- 公共耦合
- 内容耦合

#### 4. 软件测试
**Q: 软件测试的目的？**
A: 发现错误，而不是证明程序是正确的。

**Q: 黑盒测试和白盒测试的区别？**
A: 
- 黑盒测试：不考虑内部逻辑，只测功能（等价类划分、边界值分析、错误推测法、因果图）
- 白盒测试：考虑内部逻辑（逻辑覆盖、基本路径测试）

**Q: 软件测试的步骤？**
A: 
- 单元测试（模块测试）
- 集成测试（组装测试）
- 确认测试
- 系统测试

### 四、数据库设计基础

#### 1. 数据库系统的基本概念
**Q: 数据、数据库、数据库管理系统、数据库系统的定义？**
A: 
- 数据：描述事物的符号记录
- 数据库：长期存储在计算机内、有组织的、可共享的数据集合
- 数据库管理系统（DBMS）：数据库系统的核心软件
- 数据库系统（DBS）：引入数据库后的系统

**Q: 数据库系统的特点？**
A: 
- 数据独立性高（物理独立性、逻辑独立性）
- 数据共享性高，冗余度低
- 数据由DBMS统一管理和控制

#### 2. 数据模型
**Q: 常见的数据模型？**
A: 
- 层次模型（树状结构）
- 网状模型（网状结构）
- 关系模型（二维表）← 二级考试重点
- 面向对象模型

**Q: 关系模型的基本概念？**
A: 
- 关系：二维表
- 元组：表中的一行（记录）
- 属性：表中的一列（字段）
- 主键：唯一标识元组的属性或属性组
- 外键：参照其他表主键的属性

#### 3. 关系代数
**Q: 关系的基本运算？**
A: 
- 选择（σ）：从关系中选取满足条件的元组
- 投影（π）：从关系中选择若干属性列
- 连接（∞）：将两个关系拼接成一个新关系

#### 4. 数据库设计
**Q: 数据库设计的两个映射？**
A: 
- E-R图到关系模型的转换
- 关系模型到数据库表的转换

**Q: E-R图中三个基本概念？**
A: 
- 实体：矩形表示
- 属性：椭圆表示
- 联系：菱形表示（1:1、1:n、m:n）

### 二、常用函数

#### 1. 求和函数
**Q: SUM函数的用法？**
A: `=SUM(number1, [number2], ...)`  
示例：`=SUM(A1:A10)` 求和A1到A10

**Q: SUMIF函数的用法？**
A: `=SUMIF(range, criteria, [sum_range])`  
示例：`=SUMIF(A1:A10,">60",B1:B10)` 对A1:A10中大于60的对应B列求和

**Q: SUMIFS函数的用法？（多条件求和）**
A: `=SUMIFS(sum_range, criteria_range1, criteria1, ...)`  
示例：`=SUMIFS(C1:C10,A1:A10,"男",B1:B10,">60")`

#### 2. 平均值函数
**Q: AVERAGE函数的用法？**
A: `=AVERAGE(number1, [number2], ...)`

**Q: AVERAGEIF和AVERAGEIFS的用法？**
A: 与SUMIF/SUMIFS类似

#### 3. 逻辑函数
**Q: IF函数的用法？**
A: `=IF(logical_test, value_if_true, value_if_false)`  
示例：`=IF(A1>=60,"及格","不及格")`

**Q: AND和OR函数的用法？**
A: 
- `=AND(条件1, 条件2, ...)` 所有条件为真才返回真
- `=OR(条件1, 条件2, ...)` 任一条件为真就返回真

**Q: 嵌套IF的用法？**
A: `=IF(A1>=90,"优秀",IF(A1>=80,"良好",IF(A1>=60,"及格","不及格")))`

#### 4. 查找函数
**Q: VLOOKUP函数的用法？**
A: `=VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])`  
- lookup_value：要查找的值
- table_array：查找区域
- col_index_num：返回第几列的数据
- range_lookup：TRUE为模糊匹配，FALSE为精确匹配

示例：`=VLOOKUP(A1,Sheet2!A:B,2,FALSE)`

**Q: INDEX和MATCH组合的用法？**
A: `=INDEX(返回区域, MATCH(查找值, 查找区域, 0))`  
比VLOOKUP更灵活，可以从右向左查找

#### 5. 文本函数
**Q: LEFT、RIGHT、MID函数的用法？**
A: 
- `=LEFT(text, num_chars)` 从左侧截取
- `=RIGHT(text, num_chars)` 从右侧截取
- `=MID(text, start_num, num_chars)` 从中间截取

**Q: TRIM函数的用法？**
A: `=TRIM(text)` 删除文本中多余的空格

**Q: CONCATENATE或&的用法？**
A: `=CONCATENATE(text1, text2, ...)` 或 `=A1&B1`

#### 6. 日期函数
**Q: DATE函数的用法？**
A: `=DATE(year, month, day)`

**Q: YEAR、MONTH、DAY函数的用法？**
A: 
- `=YEAR(serial_number)` 提取年份
- `=MONTH(serial_number)` 提取月份
- `=DAY(serial_number)` 提取日期

**Q: TODAY和NOW函数的用法？**
A: 
- `=TODAY()` 返回当前日期
- `=NOW()` 返回当前日期和时间

#### 7. 计数函数
**Q: COUNT和COUNTA的区别？**
A: 
- `=COUNT(value1, [value2], ...)` 只计数数字
- `=COUNTA(value1, [value2], ...)` 计数所有非空单元格

**Q: COUNTIF和COUNTIFS的用法？**
A: 
- `=COUNTIF(range, criteria)`
- `=COUNTIFS(criteria_range1, criteria1, ...)`

**Q: 如何设置统一的页眉页脚？**

### 三、动画和切换

**Q: 如何添加动画？**
A: 选中对象 → 动画选项卡 → 动画组 → 选择动画效果

**Q: 动画的类型有哪些？**
A: 
- 进入动画：出现、淡入、飞入等
- 强调动画：放大/缩小、闪烁等
- 退出动画：消失、淡出、飞出等
- 动作路径：自定义路径

**Q: 如何设置动画顺序？**
A: 动画选项卡 → 高级动画组 → 动画窗格  
在动画窗格中拖动调整顺序

**Q: 如何设置动画的开始方式？**
A: 
- 单击开始：点击鼠标时开始
- 与上一个动画同时：和上个动画同时开始
- 上一个动画之后：上个动画结束后自动开始

**Q: 如何设置幻灯片切换效果？**
A: 切换选项卡 → 切换到此幻灯片组 → 选择切换效果

**Q: 如何设置自动换片？**
A: 切换选项卡 → 计时组 → 设置自动换片时间

**Q: 如何设置自定义放映？**

**Q: 快捷键：如何从当前幻灯片开始放映？**
