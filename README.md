## 🎯 备考策略

### 三阶段备考法

**第一阶段（5月中旬 - 6月底）**：基础打牢
- Word模块：样式、目录、邮件合并
- PPT模块：母版、动画、切换
- Excel入门：基本函数、排序筛选

**第二阶段（7月 - 8月上旬）**：重点突破
- Excel大魔王：VLOOKUP、数据透视表
- 选择题背诵：公共基础知识 + Office基础

**第三阶段（8月中旬 - 9月初）**：真题冲刺
- 刷历年真题
- 整理错题本
- 全真模拟

## 🚀 使用方法

- 打开本仓库的 `知识库大全.md`

### 方法二：克隆到本地

## 📝 知识库使用技巧

`知识库大全.md` 采用 **Q&A 格式**，建议这样使用：

1. **第一轮**：全部过一遍，理解为主

## 💡 备考提醒

1. **Excel是重中之重**：占分最高，必须重点突破
2. **选择题不能放弃**：20分选择题，争取拿15分以上
3. **操作题要"手熟"**：不要只看不练，每套题至少做3遍
4. **利用碎片时间**：用手机打开本仓库复习选择题
